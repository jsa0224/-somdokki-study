# ****connect()****

Publisher에 연결하여 요소를 생성하도록 허용하고 게시를 취소할 인스턴스를 반환한다.

```swift
func connect() -> Cancellable
```

# ****Return Value****

게시를 취소하는 데 사용하는 [Cancellable](https://www.notion.so/Cancellable-44842f71a8104896b71fe8ab809dbbd5?pvs=21) 인스턴스이다.
