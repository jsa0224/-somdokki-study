# Overview

[`cancel()`](https://developer.apple.com/documentation/combine/cancellable/cancel())을 호출하면 할당된 리소스가 해제된다. 
또한 타이머, 네트워크 액세스 또는 디스크 I/O와 같은 부작용을 중지한다.
