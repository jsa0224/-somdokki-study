# ****Overview****

[`Publisher`](https://developer.apple.com/documentation/combine/publisher)에서 확장으로 정의된 다양한 연산자는 이 열거형을 확장하는 클래스 또는 구조로 기능을 구현한다.
예를 들어, `contains(_:)` 연산자는 `Publishers.Contains` 인스턴스를 반환한다.
